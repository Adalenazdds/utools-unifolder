# UniFolder 📁✨

UniFolder 是一个 uTools 插件：帮你给**文件/文件夹**添加别名、标签与备注 🏷️📝，并通过标签快速筛选与查找 🔎。

一句话：把“散落在硬盘里的路径”，整理成“可检索的个人资料库”。

## 功能概览 ✅

- 📌 条目管理：对任意文件/文件夹设置别名、标签与详细备注。
- 🛡️ 备注防误触：备注为空时直接编辑；已有内容时默认预览态，点“编辑备注”才进入编辑。
- 🔗 网址自动识别：备注预览会把 URL 识别成可点击链接，并在默认浏览器打开。
- 🗂️ 全局标签库：在侧栏“编辑”模式中新增/删除标签；删除标签会自动清理所有条目中的引用。
- 🔎 标签筛选：侧栏按标签快速筛选条目。
- 🧱 固定高度标签区：标签分配区域固定高度、内部滚动，不挤占备注输入框。
- 💾 位置记忆：自动记录上次关闭时的工作目录、筛选标签与选中条目，下次打开自动恢复。
- 🫥 侧栏隐藏/显示：左侧侧栏一键切换，中间与右侧布局自适应。
- ⚡ uTools 搜索直达标签：将全局标签注册为 uTools **动态指令**，在 uTools 搜索框直接搜标签即可进入，并自动按该标签筛选。

## 最近更新（备注体验）

- 进入条目时：备注为空 → 直接可编辑；备注非空 → 默认显示预览，避免误触覆盖长文。
- 需要修改时：点击“编辑备注”立即解锁编辑（无弹窗确认）。
- 预览态支持链接点击：`http(s)://` 与 `www.` 开头的地址会被识别为链接并可打开。

## 适用场景 🎯

- “这个项目的入口文件在哪？”——给项目根目录打上 `project` / `work` 标签
- “常用工具/脚本太多记不住”——用备注记录用途与命令行
- “素材/参考链接分类混乱”——用标签把路径按主题归档

## 项目结构 🧩

- `index.html`：前端 UI（Vue 3 + Tailwind CDN）。
- `preload.js`：uTools preload 脚本（Node.js 能力、存储、动态指令注册等）。
- `plugin.json`：uTools 插件配置。
- `UniFolder_Backup.json`：示例备份（新 schema）。

## 开发/调试（Dev 模式）🛠️

1. 安装 uTools。
2. 安装「uTools 开发者工具」插件。
3. 在开发者工具中选择“加载本地插件应用”，指向本仓库目录（包含 `plugin.json`）。
4. 在 uTools 主搜索框输入 `unifolder` 启动。

> 说明：Dev 模式不会自动同步到其他电脑；如果你在多台电脑使用，需要分别加载/安装该插件。

## 数据存储与备份 💽

- 数据存储使用 `utools.dbStorage`。
- 当前数据 schema（v2）：

```json
{
  "schemaVersion": 2,
  "tagsList": ["tag1", "tag2"],
  "items": {
    "C:/path/to/item": {
      "alias": "",
      "note": "",
      "tags": ["tag1"],
      "hidden": false
    }
  }
}
```

- 导入/导出：侧栏提供“导入数据 / 导出备份”。导入兼容旧格式并自动迁移 ✅

## 标签搜索（动态指令）⚡

- 新增的全局标签会通过 `utools.setFeature()` 注册为动态指令。
- 在 uTools 搜索框直接输入标签名（或 `#标签`）即可命中并进入插件；进入后自动切换到该标签筛选视图。
- 删除全局标签会同时卸载对应动态指令（`utools.removeFeature()`）。
